\section{What you need (supplies)}

To complete this introductory course in guitar, you will need the following supplies:

\begin{itemize}
	\item A working classical (nylon-string) guitar
	\item A footstool
	\item A pencil
	\item A binder
	\item An electronic tuner (app or standalone device)
	\item An electronic metronome (app or standalone device)
\end{itemize}
